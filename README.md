## 📌 Project Overview
This repository contains the analysis and Power BI dashboard for the Event Management Data Hackathon. The project explores eight years of event data (2017-2025) across Indian cities.The objective is to uncover actionable insights on event trends, category performance, regional variations, and financial outcomes empowering organizers and stakeholders to make data-driven decisions.

## 📊 Key Insights & Analytics

### 💰 Profitability & ROI Analysis 
* **Total Revenue:** 358M 
* **Total Investment:** 227M 
* **Total Profit:** 131M 
* **Overall ROI:** 57.82% 
* **Top States by Investment:** Tamil Nadu (31M) , Maharashtra (30M), and Telangana (25M)

### 👥 Attendance & Engagement Insights
* **Total Registrations:** 537K 
* **Total Attendance:** 486K 
* **Registration to Attendance Conversion:** 90.35%
* **Average Event Duration:** 3.59 hours

### 🏢 Organizer Performance & Ratings 
* **Total Organizers:** 41 
* **Average Rating:** 4.34 
* **Average Organizer ROI:** 39.52% 
* **Top Rated Organizers:** Global Flavors (4.8) and Flour Power (4.7)
