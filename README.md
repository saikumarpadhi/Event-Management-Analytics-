## 📌 Project Overview
[cite_start]This repository contains the analysis and Power BI dashboard for the Event Management Data Hackathon[cite: 24]. [cite_start]The project explores eight years of event data (2017-2025) across Indian cities[cite: 25]. [cite_start]The objective is to uncover actionable insights on event trends, category performance, regional variations, and financial outcomes empowering organizers and stakeholders to make data-driven decisions[cite: 28].
